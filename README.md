# SVSS Website Package

This is a lightweight static website package for the Sacramento Valley Soaring Society.

## Included pages

- index.html
- schedule.html
- spring-fling.html
- fall-fling.html
- results.html
- field-info.html
- join.html
- officers.html
- gallery.html
- contact.html

## Deployment on GitHub Pages

1. Create a GitHub repository.
2. Upload all files and folders from this package to the repository root.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save. GitHub Pages will publish the site.

## Editing tips

- Global styling is in assets/css/styles.css
- Navigation appears on every page
- Update homepage event information in index.html
- Update the full season in schedule.html
- Update major event details in spring-fling.html and fall-fling.html
- Replace SVG placeholders in assets/images/ with real images when ready

## Placeholder files

Several placeholder files are included under assets/docs/. Replace them with PDFs or your actual documents as needed.
